# Sicktown Poker League Manager

## Introduction

This will be a full stack web application that will be used to manage a private poker league.

It is still in the very early stages of development so anything may break or change at any time.

## Initial Planned Features

- Player Management: Create and track players and their results.
- Season Management: Create complete poker tournament seasons, with multiple tournaments, scoring, calculate season stats and league leaders.
- Tournament Management: Allow player registration, recording and viewing of results, scoring, calculate payouts
- Invitation only access to view results or register for tournaments

## Technologies Used
 - TypeScript - Complete type safety from database to client
 - Next.js/React - Full-stack application framework
 - GraphQL - Served from Next.js API endpoint. Code-first schema creation with Pothos GraphQL
 - Database - PlanetScale, MySQL-compatible serverless database, using Prisma ORM
